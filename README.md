# Modulo-3-Reconocimiento


hice un ejercicio de reconocimiento pasivo con búsquedas en google. Tuve que usar operadores  como site:, filetype:, inurl:, intitle:, “” para frases exactas, OR y - para buscar en la web diferentes tipos de información pública. Estas búsquedas me dejaron ver cómo Google es capaz de mostrar páginas, documentos o directorios que están disponibles al público. También revisé algunos de los resultados para ver si alguno de ellos pudiera suponer algún riesgo de seguridad, como por ejemplo la aparición de archivos o directorios que pudieran contener información sensible. 

en el paso 2, entré en la Google Hacking Database, que se encuentra dentro de Exploit Database, use 3 Google Dorks, en los cuales revisé qué tipo de información buscan, qué exposición pueden detectar y en qué categoría se clasifican de peligro . 

en el último paso, utilicé Maltego para ver y crear un grafo  para entender  la relación entre dominio, DNS, IP y ubicación de un dominio
